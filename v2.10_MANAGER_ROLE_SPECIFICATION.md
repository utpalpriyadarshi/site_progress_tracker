## 👥 User Roles & Access

### Role Definitions

**All use same role type: `manager`**
Differentiated by `full_name` or `designation` field

1. **Project Manager (PM)**
   - Full project oversight
   - All sites visibility
   - All dashboard sections
   - Approval authority

2. **Deputy Project Manager (DPM)**
   - Same access as PM (Option A confirmed)
   - Can act when PM unavailable
   - Full project visibility

3. **Commercial Project Manager**
   - Same access as PM
   - View all details
   - Financial focus but not restricted

4. **Scheduler (Planning Role)**
   - Uses `planning` role (separate implementation)
   - Schedule management focus

### Access Model
```
Manager Assignment:
- One manager = one project (like supervisor in v2.9)
- Can view ALL sites under assigned project
- Database field: users.project_id (already exists from v2.9)
- Context: ManagerContext (similar to SiteContext)
```

### Team Hierarchy
```
Project
  └── Manager (PM)
       ├── DPM 1 (also manager role)
       ├── DPM 2 (also manager role)
       ├── Commercial PM (also manager role)
       └── Supervisors (multiple)
            └── Sites (assigned to each supervisor)
```

**Note:** DPM-to-Supervisor mapping NOT required. All managers see all supervisors in the project.

**3.2 Progress Calculation:**
```
Site Progress =
  (0.6 × Weighted Items Progress) +
  (0.4 × Milestone-based Progress)

Where:
- Weighted Items Progress = Σ(item_completion × item_weightage) / Σ(item_weightage)
- Milestone Progress = Σ(milestone_completion × milestone_weightage) / 100
```

## 📊 Progress Calculation Formulas

### Site-Level Progress (Hybrid - Option B + C)

```typescript
// Weighted Items Progress (Option B)
function calculateWeightedItemsProgress(items: ItemModel[]): number {
  const totalWeightage = items.reduce((sum, item) => sum + item.weightage, 0);
  if (totalWeightage === 0) return 0;

  const weightedProgress = items.reduce((sum, item) => {
    const itemProgress = (item.completedQuantity / item.plannedQuantity) * 100;
    return sum + (itemProgress * item.weightage);
  }, 0);

  return weightedProgress / totalWeightage;
}

// Milestone-Based Progress (Option C)
function calculateMilestoneProgress(milestoneProgress: MilestoneProgressModel[]): number {
  const totalWeightage = milestoneProgress.reduce(
    (sum, mp) => sum + mp.milestone.weightage,
    0
  );
  if (totalWeightage === 0) return 0;

  const weightedProgress = milestoneProgress.reduce((sum, mp) => {
    return sum + (mp.progressPercentage * mp.milestone.weightage);
  }, 0);

  return weightedProgress / totalWeightage;
}

// Hybrid Site Progress
function calculateSiteProgress(
  items: ItemModel[],
  milestoneProgress: MilestoneProgressModel[]
): number {
  const itemsProgress = calculateWeightedItemsProgress(items);
  const milestonesProgress = calculateMilestoneProgress(milestoneProgress);

  // 60% weight to items, 40% weight to milestones
  return (0.6 * itemsProgress) + (0.4 * milestonesProgress);
}
```

### Project-Level Progress

```typescript
function calculateProjectProgress(sites: SiteModel[]): number {
  if (sites.length === 0) return 0;

  const totalProgress = sites.reduce((sum, site) => {
    return sum + calculateSiteProgress(site.items, site.milestoneProgress);
  }, 0);

  return totalProgress / sites.length;
}
```
