## 📊 Main Feature: BOM Management System

### Overview
Dual-mode BOM system supporting both Pre-Contract (Estimating) and Post-Contract (Execution) workflows with variance tracking and professional reporting.

### Core Features

#### 1. **Dual BOM Types**
**Pre-Contract BOMs (Estimating):**
- Status flow: Draft → Submitted → Won/Lost
- Used for tender preparation
- Becomes baseline when project is won

**Post-Contract BOMs (Execution):**
- Status flow: Baseline → Active → Closed
- Tracks actual costs vs. estimates
- Links to baseline for variance analysis

#### 2. **Site Categorization**
7 Metro Railway site types:
- 🚇 ROCS - Receiving Overhead Catenary System
- ⚡ FOCS - Feeding Overhead Catenary System
- 🔌 RSS - Receiving Sub-Station
- 🏗️ AMS - Ancillary Multi-function Structure
- 🔋 TSS - Traction Sub-Station
- ⚙️ ASS - Auxiliary Sub-Station
- 🌉 Viaduct - Elevated Structure

#### 3. **Item Management**
4 item categories with auto-generated codes:
- 📦 **Material** (MAT-001, MAT-002...) - Physical materials
- 👷 **Labor** (LAB-001, LAB-002...) - Work hours
- 🔧 **Equipment** (EQP-001, EQP-002...) - Machinery rental
- 🏢 **Subcontractor** (SUB-001, SUB-002...) - Third-party work

**Features:**
- Item code, name, description
- Quantity with units (m, kg, hrs, nos, etc.)
- Unit cost with automatic total calculation
- Category-based color coding
- Real-time cost aggregation

#### 4. **Variance Tracking**
- Link execution BOMs to baseline BOMs
- Automatic variance calculation (Actual vs. Baseline)
- Percentage and absolute variance display
- Color-coded indicators:
  - 🔴 Red = Cost overrun (bad)
  - 🟢 Green = Cost savings (good)
  - ⚪ Gray = No variance

#### 5. **Excel Export**
Professional 2-sheet Excel export:
- **Sheet 1 (Summary):** BOM metadata, total cost, variance
- **Sheet 2 (Items):** All line items with calculations
- Auto-sized columns
- Formatted for client presentations
- Saves to Downloads folder

### Database Schema (v21-v25)

**boms table:**
```typescript
id, name, type, status, site_category, site_id, project_id,
created_by_id, created_at, updated_at, total_cost,

```

**bom_items table:**
```typescript

quantity, unit, unit_cost, total_cost, created_at, updated_at
```
